GitHub Dashboard :

Dashi

GitHub Dashboard "Dashi" is a web application that provides detailed statistics about a GitHub user's profile and repositories.

About

Insperation

The inspiration for the GitHub Dashboard project came from my experience using the GitHub platform. I found myself fascinated by the wealth of data available on users and repositories, from activity stats to the variety of programming languages used. I wanted to build my own dashboard that could take this data and present it in a user-friendly wayâ€”allowing anyone to explore GitHub users, their repositories, and the technologies behind them. The project grew out of a desire to make it easier for developers to understand their own and others' coding habits and repository stats at a glance.

Additionally, I plan to expand the dashboard in the future to include features that track contributions and user activities, offering a more comprehensive view of GitHub profiles and project dynamics.

Project features :

Login Page

Enter a GitHub username to view the user's profile data.

Home Page

Displays user profile statistics, including:

  • Username
  • Profile picture
  • Number of public repositories
  • Number of public gists
  • Number of followers
  • Number of following

Sidebar Navigation

A sidebar feature helps you navigate through:

  • Home Page
  • Repositories Page (table)

Search Bar

Search for a specific repository from the list of the user's repositories.

Repository Statistics

Displays detailed statistics for each repository, including:

  • Languages used
  • Number of issues
  • Stars
  • Watchers
  • Forks
